# Workflow Specification

This module defines the workflow "language" used by MedFlow Studio.

A workflow describes:
- Data sources
- Transformations
- Destinations
- APIs
- How these blocks are connected

This specification is:
- Generated by the frontend (no-code UI)
- Consumed by the backend compiler
- Independent of UI, backend, and AI

No code in this folder executes anything.
It only defines structure and rules.
